# 聖境塔羅 (Celestial Tarot)

探索靈魂的指引：以純前端打造的沉浸式塔羅抽牌體驗，整合手勢辨識、AI 綜合神諭、PWA 與結果圖片匯出。

目前版本：`v1.4.16`

## 核心功能

- **3D 抽牌互動**
  - 扇形景深卡牌圓環與流暢旋轉動畫。
  - 卡牌翻面後飛入三個結果槽位，保留正逆位視覺。

- **提問導向的 AI 綜合神諭**
  - 支援 `Gemma 4 31B`（預設）與 `Gemini 3 Flash`。
  - AI 會依「使用者提問」動態定義三張牌角色，再產出整體解讀（不是固定過去/現在/未來）。
  - 內建全螢幕等待畫面與打字機顯示效果。

- **手勢與一般操作並行**
  - 一般操作可用滑鼠／觸控完成抽牌。
  - 啟用手勢後支援：
    - 張開手掌：旋轉卡環
    - 握拳：停止並鎖定前方卡牌
    - 食指上滑：確認選牌

- **解牌結果圖片輸出**
  - 可匯出「提問 + 所選三張牌 + 星辰指引」為 PNG 圖片。
  - 下載檔名會帶時間戳，方便留存與分享。

- **PWA 與裝置友善**
  - 可安裝為桌面/手機應用、支援快取離線資源。
  - 支援 Wake Lock，降低抽牌過程螢幕休眠機率。
  - 偵測 LINE / Facebook 內建瀏覽器並提示改用外部瀏覽器。

## 資料來源與回退機制

- 啟動時會嘗試從 `tarotapi.dev` 載入完整 78 張牌資料。
- 若 API 載入失敗，會自動回退到內建的 22 張大阿爾克那資料，確保流程可用。

## 快速開始

本專案為 Vanilla HTML/CSS/JavaScript，無需安裝建置工具。

1. 啟動本地伺服器（不要直接雙擊 `index.html`）：

```bash
python -m http.server 8000
```

或

```bash
npx http-server . -p 8000
```

2. 開啟 `http://localhost:8000`。
3. 如需 AI 解讀，點右上角設定，填入 Google AI API Key 並選擇模型。

## 專案結構

- `index.html`：主要頁面結構、設定與解牌 Modal。
- `app.js`：抽牌流程、手勢控制、AI 串接、圖片輸出、狀態管理。
- `style.css`：整體視覺、動畫、響應式版型。
- `tarot_dict.js`：塔羅牌中文資料與牌意翻譯。
- `version.js`：版本號常數（用於快取更新）。
- `manifest.json`、`sw.js`：PWA 設定與 Service Worker。
- `images/`：卡牌圖片與畫面素材。
- `icons/`：PWA 圖示。

## 常見問題

- **手勢無法使用**
  - 先確認已授權相機權限，並等待手勢模型初始化。
  - 若在 LINE / Facebook 內建瀏覽器，請改用外部瀏覽器開啟。

- **AI 無回應或失敗**
  - 確認 API Key 正確、網路可用、模型配額未超限。
  - 未設定 API Key 時，系統會回傳提示文字而非完整 AI 神諭。

- **無法儲存星辰指引圖**
  - 需先完成一次解牌並產出指引內容後才能匯出。
  - 若仍失敗，請重試或更換瀏覽器。
